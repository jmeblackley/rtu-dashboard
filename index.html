    About this dashboard

    Automated detection of rooftop HVAC units (RTUs) over Moncton, New Brunswick, from 7.5 cm aerial imagery (GeoNB, 2024).

    Which buildings are included

    Analysis covers “Part 3” buildings — footprints larger than 600 m² (the larger commercial and institutional roofs where RTUs are expected). Smaller buildings aren’t analysed, so they don’t appear on the map or in search.

    Addresses & search

    Addresses come from the building-footprint dataset’s attributes. Search only covers the analysed Part 3 buildings, and not every footprint carries a complete address — so a valid street address can return nothing if its building falls outside this set or is unlabelled. Try a nearby larger building, or pan the map and click a footprint directly.

    How detections are made

    A deep-learning object-detection model draws a bounding box around each likely RTU and reports a confidence score (85–100%). Use the confidence filter to show only higher-confidence detections.

    How accuracy was measured

    Detections were compared against 2,415 hand-labelled RTUs on 508 buildings using the PASCAL VOC method (a match requires IoU ≥ 0.5). Result: 89.4% recall (real RTUs found) and 89.7% precision (detections that are correct). The validation layer maps each case as correct (TP), false positive (FP) or missed (FN).
